## To Creat emotion-Diary(감정일기장) with React😊

### React router

기존의 a 태그를 활용한 페이지 라우팅 방식과는 다르게 SPA를 사용하는 React의 라우팅을 활용

#### React Router 응용

1. path Variable
   useParams라는 React router의 커스텀을 이용

2. Query
   name과 value를 엮어서 사용

3. Navigate
   의도적으로 페이지를 바꿀 수 있음. ex. 로그인 안 했을 때 등등..

4. useReducer, createContext 사용
